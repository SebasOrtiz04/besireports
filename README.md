# Generador de reportes besi para surtido de cajas

## Descripción

Este proyecto es una aplicación de escritorio para windows desarrollada en Python utilizando Tkinter. 

La aplicación permite al usuario cargar dos archivos CSV, uno con la información provenitente del Besi comparar sus datos, filtrar la información relevante, realizar cálculos y generar reportes en formatos PDF y Excel

- Reporte formtato excel .xlsx surtido de cajas:
- Reporte formato pdf .pdf separado por plataforma y por surtidor
